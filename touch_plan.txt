### 3) Linux touch daemon
- Keep `touch_daemon.py` as the Linux input service.
- Use `snegg.oeffis` to do the portal handshake and request the RemoteDesktop session.
- Use `snegg.ei` for the libei client connection to the compositor.
- Keep the touch capability bound as `TOUCHSCREEN` so the real mouse cursor does not move.
- Keep the TCP server on `127.0.0.1:7111`.
- Keep the Linux side responsible for scaling normalized Android coordinates back to the monitor resolution and sending Wayland input frames through libei. ### 4) Event handling fix
- Do **not** assume snegg’s Python `EventType` enum is exhaustive.
- Handle raw libei event types safely so newer values like `SYNC` and `PONG` do not crash the daemon.
- Treat unknown control events as ignorable and continue the loop.
- Continue waiting until the daemon sees the seat/device events needed to finish setup. [libinput.pages.freedesktop](https://libinput.pages.freedesktop.org/libei/api/group__libeis.html)
